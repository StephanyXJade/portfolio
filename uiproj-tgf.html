THE GREAT FROG WEBSITE

The Great Frog is a premier jewelry e-commerce platform known for its one-of-a-kind designs inspired by rock culture. A wide variety of merchandise, including collaborations with prestigious brands, are proudly shown on our website. The main challenge in this website is addressing the issue of how complex is to find items browsing through the website. It usually takes several clicks, frequently taking longer than intended.

BACKGROUND

The website's current design has significant usability problems. It is difficult to find the shop because there are no clear guidelines between the sections or elements. The contrast between titles, calls-to-action, and other elements becomes confused since there is not much space between sections. In addition, a lack of visual hierarchy is a result of the absence of accent colours and contrast.

PROBLEM

The icons library doesn't effectively balance the importance between icons and text, and the content feels crowded, resulting in an overly busy and cluttered website interface. Together, these problems make it difficult for users to access the shop easily, which lowers user engagement and satisfaction.

DESIGN PROCESS

USER PERSONA

WEBSITE ANALYSIS

User needs to hover over shop button to see the categories. There’s no store homepage. The shop option only includes a percentage of the products in stock.

Once the user gets to the store, the content is already filtered by category. The user needs to go back to the home page and hover again to change category. There’s no possibility to select multiple categories, sub-categories, products, and products related to weddings, collaborations and more. The user needs to go to another page to purchase those products.

When navigating from a mobile device, the navigation menu changes slightly. There are two more buttons (new and ready to ship). On the desktop layout, these buttons appear as sub-pages below the Shop option. In addition, if the user previously selected a category and returns to change to another, the nav menu still displays subcategories from the previews category.

SOLUTION

Content Hierarchy

Reorganize the website's content to establish a clear hierarchy. Prioritize essential sections such as Shop, Collaborations, and New Designs, followed by supporting information like About Us, Contact, and Blog. This ensures that users quickly find the store and relevant details without unnecessary distractions.

Guidance for Store Location

My main goal is to redesign the general shop layout. Implement a clear and intuitive 'Shop' button in the top navigation bar or a dedicated section on the homepage hero banner. Additionally, include subtle prompts within relevant content areas guiding users towards the shop. (Rearrange the sitemap flow so the user sees the store first). Implement a side menu for adding the categories, sub-categories, filters, multiple selection components and include the stock from collabs, deluxe and new pages.

Optimized Copy Content

Engaging, and simple copy following UX writing best practices. Use clear headings, descriptive subheadings, and compelling CTAs to guide users through the website. For instance, employ phrases like "Explore Our Unique Designs" and "Discover Exclusive Collaborations" to entice users and direct them towards the shop.

Enhanced Icons Library

Revise the icons library to align with the brand's aesthetics and emphasize the store. Opt for clean and visually appealing icons that resonate with the rock culture and jewelry theme. Reduce text where possible, ensuring make the site’s design lighter, giving more focus on the store's presence and products.

Forms

Enhance the forms component due to the lack of labels and accessible rules.
