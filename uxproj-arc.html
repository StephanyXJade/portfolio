ARC APP

An application designed specifically for film production companies to efficiently manage data and track the time spent by team members during shooting or work activities. ARC app, aims to simplify the production tracking process, providing simple forms to easily input real-time information, and enable data visualization for better decision-making. Gone are the days of manual reporting. Arc is a producer’s report companion, proficient in performing all calculations for you. Experience a clean interface and data visualization that offers a seamless glance at the process.

BACKGROUND

5. SYSTEM ARCHITECTURE

Our web application was created with Next.js, a Full-stack web framework. For user authentication, Clerk Auth handles login and security. Data storage relies on MongoDB as the database, with Prisma ORM acting as the intermediate layer for smooth database interaction. Prisma simplifies access and provides a secure API. The application is deployed on Vercel, a cloud platform known for being serverless and ensuring reliable availability and fast deployment. Overall, architecture offers a strong foundation for developing a modern, scalable, and secure web application.
